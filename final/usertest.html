Observations

Some observations I found while doing the user testing was:

  1. People had a hard time differentiating the “rules” page and the “help page
  2. Users had a harder time distinguishing which players turn it is
  3. It wasn't clear when a person won

Implementation

Based on the observations, I plan to implement these changes respectfully:

  1. Making the “help” page an overlay vs a page that takes up the entire screen
  2. Changing the color of the name to show whose turn it is, decreasing the opacity more for the player that isn't rolling
  3. Change the loser's saturation to black and white, having a glow effect on the background
